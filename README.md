## Integração Continua com Jenkins e Sonarqube (docker-compose)

## No diretório que se encontra o arquivo docker-compose.yml execute o comando abaixo:
`docker-compose up -d`

## Browser SonarQube
http://localhost:9002/

## Access
`
    User: admin
    Password: admin
`

## Jenkins
http://localhost:8081/
